# Maestro Task

O objetivo do sistema é auxiliar o usuário na organização de suas tarefas de forma simples, prática e eficiente. O Maestro Task permite o cadastro, a visualização e o acompanhamento das tarefas, facilitando o controle das atividades do dia a dia.

## ⚙️ Funcionalidades

O sistema possui as seguintes funcionalidades principais:
* Criar tarefas
* Listar tarefas
* Marcar tarefas como concluídas
* Excluir tarefas

## 🛠️ Tecnologias Utilizadas

O sistema foi desenvolvido utilizando tecnologias básicas de desenvolvimento web, sem o uso de frameworks ou bibliotecas externas:
* **HTML5:** utilizado para estruturar a interface do sistema.
* **CSS3:** responsável pela estilização da interface, definição de layout, cores e responsividade básica.
* **JavaScript (ES6+):** responsável pela lógica do sistema, incluindo criação, listagem, atualização e exclusão de tarefas.

## 🚀 Como executar o projeto

Como o projeto é uma Single Page Application (SPA) simples, não é necessária nenhuma instalação complexa:
1. Faça o clone deste repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` diretamente no seu navegador de preferência (Chrome, Firefox ou Edge).
3. Adicione suas tarefas e comece a utilizar de forma simples e prática!
